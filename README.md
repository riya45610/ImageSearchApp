# 📸 Image Search App

An Image Search Application built using **HTML5, CSS3, and JavaScript** that allows users to search for high-quality images using an external image API. The application provides a clean, responsive interface and displays search results instantly based on user input.

## 🚀 Features

- 🔍 Search images by keyword
- 🖼️ Display high-quality image results
- 📱 Responsive design for desktop and mobile
- ⚡ Fast API integration using JavaScript Fetch API
- 🔄 Load multiple search results dynamically
- 🎨 Simple and user-friendly interface

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- REST API (Image Search API)
- Fetch API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/riya45610/image-search-app.git
```

2. Navigate to the project directory

```bash
cd image-search-app
```

3. Open `index.html` in your browser.

## 🎯 How It Works

1. Enter a keyword in the search box.
2. Click the **Search** button.
3. The application sends a request to the Image Search API.
4. Matching images are displayed on the screen.
5. Users can perform multiple searches without refreshing the page.
